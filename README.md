# BMW-Global-Sales-Dashboard-2018-2025-
An interactive data visualization dashboard built with D3.js that explores BMW's global sales performance from 2018 to 2025. The dashboard features a sleek dark-themed UI with glassmorphism styling and provides insights across multiple dimensions including regional sales, model performance, revenue trends, etc.

**Features**

Interactive KPI cards with real-time filtering
Multiple chart types (line, bar, area, etc.) powered by D3.js
Filter by year, region, and model
CSV data upload support for custom datasets
Responsive grid layout with hover animations and tooltips
Dark mode design inspired by BMW's brand identity
Tech Stack: HTML, CSS, JavaScript, D3.js

**Dataset:**
Monthly BMW sales data covering units sold, average price, revenue, BEV share, premium share, GDP growth, and fuel price index across Europe, China, and other global markets.
